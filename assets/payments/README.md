# Logos des moyens de paiement

Ce dossier contient les logos des moyens de paiement utilisés dans l'application.

## Fichiers requis :

1. **wave-logo.png** - Logo Wave (mobile money)
   - Format recommandé : PNG avec fond transparent
   - Taille recommandée : 200x200px minimum
   - Couleurs : Logo Wave officiel (bleu #1E88E5)

2. **orange-money-logo.png** - Logo Orange Money
   - Format recommandé : PNG avec fond transparent  
   - Taille recommandée : 200x200px minimum
   - Couleurs : Logo Orange Money officiel (orange #FF6600)

## Instructions :

⚠️ **IMPORTANT** : Les fichiers `wave-logo.png` et `orange-money-logo.png` dans ce dossier sont actuellement des placeholders transparents (1x1px). Vous devez les remplacer par les vrais logos !

1. Téléchargez les logos officiels de Wave et Orange Money
2. Renommez-les respectivement en `wave-logo.png` et `orange-money-logo.png`
3. Remplacez les fichiers placeholder dans ce dossier (`assets/payments/`)
4. Assurez-vous que les fichiers ont des fonds transparents pour un meilleur rendu
5. Taille recommandée : 200x200px minimum, format PNG

## Sources pour les logos :

- **Wave** : https://www.wave.com ou contactez Wave pour obtenir leur logo officiel
- **Orange Money** : https://www.orange.com ou contactez Orange pour obtenir leur logo officiel
